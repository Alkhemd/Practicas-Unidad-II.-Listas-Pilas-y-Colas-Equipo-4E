Ejercicio 1: Invertir Palabra

Usando una pila de caracteres

📋 Descripción del Ejercicio

Este ejercicio utiliza una pila (Stack) para invertir el orden de los caracteres de una palabra. Cada carácter se apila (push) y luego se desapila (pop) para obtener el orden inverso.

💡 Ejemplo: "HOLA" → "ALOH"
